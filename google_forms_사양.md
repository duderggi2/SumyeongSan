## 2. 폼 설명 (제목 바로 아래, 그대로 복붙)

```
서울 강서구 화곡동 산105-10번지 (1,636㎡, 약 495평)는 강서구청이 2001년부터 25년간 무단으로 사용해 온 사유지입니다.

법원은 1심·항소심·대법원에서 모두 강서구청의 불법점유를 인정하고 「철거 + 토지 인도 + 부당이득 지급」을 명령하였습니다 (2025.3.13. 확정).

그러나 강서구청은:
- 법원 화해권고(1억 + 연 1,400만원)를 「이유 없이」 거부했습니다
- 15년간 비과세였던 본 토지에 2026년 4년치 재산세 6,170,800원을 일괄 소급 과세했습니다
- 정당한 매수 절차 없이 매년 적은 손해배상금만 지급하며 영구 점유를 시도하고 있습니다

이는 공권력이 사인의 토지를 영구 무단 점유하는 위험한 선례가 됩니다.

본 서명은 다음 두 가지를 요청합니다:
① 강서구청은 본 토지를 정당한 가격으로 매수해야 합니다
② 강서구청의 15년 비과세 후 4년 일괄 소급 과세는 취소되어야 합니다

5분이면 끝나는 서명입니다. 본명·연락처는 받지 않습니다.
당신의 서명은 조세심판원 심판청구 시 「공익적 관심 사안」 입증 자료로 활용됩니다.

자세한 사실 관계: https://duderggi2.github.io/SumyeongSan/
```

## 3. 질문 1 — 개인정보 동의 (필수)

**유형**: 단일 선택 (라디오)
**제목**:
```
[필독] 개인정보 수집·이용 안내 — 동의 여부
```
**설명** (질문 아래 작은 글씨):
```
· 수집 항목: 닉네임 또는 이름 (필수), 거주지 / 이용 경험 / 의견 (선택)
· 수집 목적: 조세심판원 심판청구 시 공익적 관심 사안 입증, 강서구청에 시민 의견 전달
· 보유 기간: 본 분쟁 종결 시점부터 1년 후 영구 삭제
· 거부 권리: 동의를 거부하실 수 있으나, 거부 시 본 서명에 참여하실 수 없습니다
· 본명 입력 권장 X — 닉네임으로 참여 가능합니다
· 연락처는 일체 받지 않습니다
```
**선택지**:
- 동의합니다

**필수**: ✅ ON

## 8. 질문 6 — 동참 이유 (선택, 다중 선택)

**유형**: ⚠️ **체크박스** (다중 선택)
**제목**:
```
서명에 동참하시는 이유 (해당 사항 모두 선택)
```
**선택지**:
- 사유재산권의 헌법적 보호가 중요하기 때문
- 행정의 일관성과 신뢰가 중요하기 때문
- 시민에게 좋은 광장 기능은 정당한 매수로 유지되는 것이 옳기 때문
- 「벌금만 내면 영구 점유」가 가능한 선례가 만들어지면 안 되기 때문
- 보복성 행정·소급 과세에 반대하기 때문
- 기타 (직접 입력)

**필수**: ❌ OFF

## 11. 마지막 — URL 받아서 홈페이지에 붙이기

1. Google Forms 우상단 「보내기」
2. 「링크」 아이콘 (🔗) 선택
3. 「URL 단축」 체크박스 ON
4. 단축 URL 복사
5. cowork에서 `참고자료/참11_웹사이트_index_html/index.html` 열기
6. `GOOGLE_FORMS_URL_여기에_붙여넣기` 검색 → 복사한 URL로 교체
7. GitHub Pages에 푸시
8. 끝

## 향후 변경할 때 (이 문서 사용법)

폼을 변경하고 싶을 때:

- 단, 큰 변경(질문 추가·삭제)은 새 폼 만드는 게 안전 (기존 응답 보존)
- 변경 사항은 이 파일을 갱신해서 기록 보존
